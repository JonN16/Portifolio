/ projeto

GameCube

Cubo interativo feito com Arduino, Bluetooth, sensor de movimento (MPU6050) e uma matriz de LEDs 16x16, com estrutura em MDF cortada a laser (modelada em Blender). Roda Snake, Tetris, Pong e Breakout, controlados por um app Android feito em React Native. Projeto em equipe com 4 integrantes: João Eduardo, Nicolas Palácio, Lorenzo Oliveira e Mateus Bedim.

C++ Arduino React Native JavaScript Modelagem 3D

/ minha contribuição

Desenvolvi o jogo Tetris (lógica e renderização na matriz de LEDs) e o aplicativo mobile de controle em React Native.

/ como funciona

  • Estrutura em MDF cortada a laser, modelada em Blender
  • Arduino controla a matriz de LEDs 16x16 e recebe comandos via Bluetooth
  • Sensor de movimento (MPU6050) permite controles por movimento
  • App Android em React Native com um controle dedicado para cada jogo

/ jogos

Snake

Clássico jogo da cobrinha. Arraste o dedo na tela do app para controlar a direção; a cobra cresce a cada fruta capturada.

Tetris meu

Peças caem e podem ser giradas e movidas pelos botões do app; linhas completas são eliminadas. Jogo que eu desenvolvi (lógica + renderização).

Pong

Recriação do clássico Pong, com o paddle controlado pelos botões do aplicativo.

Breakout

Quebre os blocos exibidos na matriz de LEDs controlando o paddle pelo aplicativo.

/ aplicativo

Conexão com um aplicativo Android desenvolvido em React Native, integrando funcionalidades nativas (Bluetooth) e garantindo uma experiência fluida e responsiva. Cada jogo tem sua própria tela de controle dentro do app.
